Definición de Sistema Operativo

¿Qué es el sistema operativo? El sistema operativo es el software que coordina y dirige todos los servicios y aplicaciones que utiliza el usuario en una computadora.

Sistemas Operativos

Los sistemas operativos pueden ser privados o libres. Los sistemas privados tienen un código que no está disponible a modificaciones públicas y su uso es restringido, un código libre es aquel que está disponible para ser modificado

Mantenimiento preventivo de una PC

Eliminación de programas y procesos que no usamos, eliminar archivos basura, optimización del espacio del disco duro, actualización del sistema operativo, limpieza de la computadora, entre otras.
